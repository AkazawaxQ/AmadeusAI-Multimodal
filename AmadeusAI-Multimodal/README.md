## ASL Dataset & Training Information

The ASL hand sign dataset used in this project was fully collected and created by me using my own images.

Due to dataset size and ownership, the full dataset and training outputs are not included in this repository.

### Included
- Dataset structure documentation
- Small sample images for demonstration

### Not Included
- Full dataset
- Trained model weights
- Training logs (runs)

These materials are available upon request.
